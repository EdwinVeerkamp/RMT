• Wofür?

    Die Website unterstützt Fußball-Teams bei der Verwaltung von Spielen und Trainings.

    Der Team-Manager stellt Termine ein und versendet Einladungen, auf die die Spieler antworten können. Der Manager kann so jederzeit feststellen ob genug Spieler für den Termin zurückgemeldet sind.

    Weitere Features:

    • Social: Kommentarfunktion, News Feed
    • Google Maps Integration inkl. Routenplanung zu Spielorten

  • Warum?

    Das Projekt ist eine Fortführung eines alten PHP-Skripts, das mittlerweile in die Jahre gekommen ist. Das Skript wurde seinerzeit für die Freizeitliga-Mannschaft Juventus Urin (sic!) entwickelt.

    Neben der Bespaßung der Benutzer dient das Projekt v.a. dem Autor als Spielwiese für die Evaluierung und den Einsatz von Web-Technologien.

  • Beta?

    • Darstellung optimiert für Firefox.
      Andere Browser (insb. Internet Explorer) können Bereiche der Website z.T. nicht korrekt darstellen.
    • Features kommen und gehen
    • Fehler können jederzeit auftreten
    • Die Website kann jederzeit offline sein
